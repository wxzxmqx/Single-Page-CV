# Single-Page-CV
A semantically structured, SEO-optimized single-page digital CV built as part of a Frontend Development assessment. This project serves as a foundational HTML structure ready for future CSS styling.


## Project Structure

- `index.html` - The core structure and content.
- `styles.css` - (Future) Styling for the CV layout.
- `briefcase.svg` - Site icon for browser tabs.

# Tech Stack
- HTML5
- CSS3

## Usage

To view the CV locally:
1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
